# Quizine

> Knowledge, served fresh.

AI-powered certification study app. Paste or upload your study material, pick a certification, and get flashcards and practice exams generated by Claude — tailored to the official topic domains and exam weightings of your cert.

## Features

- **Flashcards & MCQ exams** generated from your own study material
- **Verified cert catalogue** with official topic domains and weightings (Databricks ML Associate/Professional)
- **Custom cert support** — enter any certification name for freeform generation
- **Spaced repetition** study sessions using the SM-2 algorithm
- **Timed exam mode** with per-domain score breakdown
- **Anki export** for offline study

## Stack

- **Backend** — FastAPI, Python 3.12, SQLAlchemy 2.0, Alembic, PostgreSQL
- **AI** — Anthropic Claude API (streaming SSE)
- **Frontend** — React, TypeScript, Tailwind CSS, Vite
- **Infrastructure** — Docker Compose

## Getting started

```bash
# 1. Clone and configure
cp .env.example .env
# Fill in ANTHROPIC_API_KEY in .env

# 2. Start the API and database
docker compose up --build

# 3. Start the frontend (separate terminal)
cd frontend && npm install && npm run dev
```

- API: http://localhost:8000
- API docs: http://localhost:8000/docs
- Frontend: http://localhost:5173

## Environment variables

| Variable | Description |
|---|---|
| `ANTHROPIC_API_KEY` | Your Anthropic API key |
| `DATABASE_URL` | PostgreSQL connection string |
| `UPLOADS_DIR` | Local path for uploaded PDFs |
| `CORS_ORIGINS` | Comma-separated allowed origins |
